## The one rule

**Don't fire any external action before the HN post is on the front page.** Every other channel is a multiplier of HN traction. Fire the multipliers in order, gated on HN performance.

## Launch in three sentences

1. Tuesday 8am ET, post Show HN with the title and body from [`SHOW-HN.md`](SHOW-HN.md). Post the first-comment seed within 60 seconds.
2. If it lands on the front page, fan out: tweet thread → LinkedIn → Discord → Reddit, in that order, spaced over 4 hours.
3. After 24h of HN traction, file the GitHub feature request and DM Boris Cherny. Stop after one follow-up at D+7.
